# Quote-Generate

Website generator quote yang sekarang dapat mengambil quote langsung dari internet menggunakan API publik dan menyesuaikannya berdasarkan kebutuhan pengguna.

## Fitur
- Filter berdasarkan tujuan quote (motivasi, sukses, kehidupan, dsb.)
- Pencarian kata kunci untuk quote yang lebih spesifik
- Pilihan jumlah quote yang ingin ditampilkan
- Tombol untuk menambahkan hasil tambahan secara berurutan
- Fallback quote lokal jika akses internet tidak tersedia

## Cara pakai
1. Buka `index.html` di browser atau jalankan server statis.
2. Pilih tujuan quote dan/atau masukkan kata kunci.
3. Klik `Ambil Quote` untuk menampilkan hasil terbaru.
4. Gunakan `Tambah Lebih Banyak` untuk menambah quote ke daftar.

## Catatan
- Mengandalkan API publik `api.quotable.io`
- Performa terbaik saat koneksi internet aktif
